# 📌 Πίνακας Βασιλέων & Πρωθυπουργών Ελλάδας
 
|   |   |   |   |
|---|---|---|---|
|**Χρόνια**|**Βασιλιάς**|**Πρωθυπουργός (ενδεικτικοί)**|**Κύρια γεγονότα – Απολογισμός**|
|1832–1862|Όθων|Ι. Κωλέττης, Αλ. Μαυροκορδάτος|**Έδωσε Σύνταγμα** (1844), κυβέρνησε απολυταρχικά, εκθρονίστηκε (1862).|
|1863–1913|Γεώργιος Α΄|Χ. Τρικούπης, Θ. Δηλιγιάννης|**Σταθεροποίησε το κράτος**, ένωσε τα Επτάνησα (1864) και τη Θεσσαλία (1881), **εκσυγχρόνισε την Ελλάδα**, δολοφονήθηκε (1913).|
|1913–1917, 1920–1922|Κωνσταντίνος Α΄|Ελ. Βενιζέλος|Διαφώνησε με τον Βενιζέλο (**Εθνικός Διχασμός**), μπήκε στον Α΄ Παγκόσμιο, **έχασε τη Μικρασιατική Εκστρατεία**, παραιτήθηκε.|
|1917–1920|Αλέξανδρος Α΄|Ελ. Βενιζέλος|Συμμάχησε με την Αντάντ, **κέρδισε νέα εδάφη** (Θράκη, Σμύρνη), πέθανε ξαφνικά.|
|1922–1924|Γεώργιος Β΄|Ν. Πλαστήρας, Αλ. Παπαναστασίου|Αντιμετώπισε τη Μικρασιατική Καταστροφή, έχασε τον θρόνο, κατήργησε τη μοναρχία (1924).|
|1935–1947|Γεώργιος Β΄ (ξανά)|Ι. Μεταξάς, Εμμ. Τσουδερός|Στήριξε τον Μεταξά, νίκησε στον πόλεμο με την Ιταλία (1940), έφυγε με την κυβέρνηση στο εξωτερικό, επέστρεψε μετά τον πόλεμο.|
|1947–1964|Παύλος|Κ. Καραμανλής, Γ. Παπανδρέου|Στήριξε την ανοικοδόμηση, έβαλε την Ελλάδα στο ΝΑΤΟ (1952), έζησε πολιτική αστάθεια.|
|1964–1973|Κωνσταντίνος Β΄|Γ. Παπανδρέου, αργότερα η χούντα|Συγκρούστηκε με τον Παπανδρέου («Αποστασία»), ανέχτηκε το πραξικόπημα (1967), έφυγε στο εξωτερικό.|
|1973–1974|— (Κατάργηση μοναρχίας)|Δικτατορία Ιωαννίδη – Κ. Καραμανλής|Έχασε την Κύπρο (1974), κατέρρευσε η χούντα, ο Καραμανλής επανέφερε τη Δημοκρατία.|
