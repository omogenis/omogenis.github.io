# Рассмотрение заявления

После прохождения собеседования пакет документов отправляется в Министерство внутренних дел Греческой Республики. 

## Проверка статуса заявления в системе Генерального секретариата по делам гражданства

Посмотреть статус дела можно на сайте Генерального секретариата по делам гражданства Министерства внутренних дел Греческой Республики.

![Пример заполнения формы для проверки статуса заявления](@site/static/img/ypes-example.png)

:::danger Внимание!

Вводите **2-3 буквы фамилии и имени на греческом языке** в той транслитерации, как они были представлены в вашем заявлении, **ЗАГЛАВНЫМИ буквами**!

:::

### Статусы рассмотрения

- **Αναμονή εξέτασης του αιτήματος με βάση το χρόνο κατάθεσης** — ожидание рассмотрения дела в соответствии с порядком очереди
    
- **Διαδικασία εξέτασης της αίτησης** — процесс рассмотрения заявления

- **Αποδοχή αίτησης** — вынесено решение

- **Αναμονή δημοσίευσης απόφασης στο Εθνικό Τυπογραφείο** — ожидание публикации решения в Правительственной газете Греческой Республики
